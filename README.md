# Student Score Manager

A Java console app to manage student scores.

## Features
- Add and view student scores
- Search by name
- Sort by score
- Find top scorer

## DSA Concepts Used
- ArrayLists
- Linear Search
- Sorting Algorithms

```bash
javac com/sarvagna/dsa/*.java
